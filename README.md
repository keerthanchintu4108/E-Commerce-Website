# 🛒 Simple E-Commerce Website

A responsive e-commerce website built using **HTML, CSS, and JavaScript**. This project demonstrates a basic online shopping interface including user authentication, product browsing, and cart management.

---

## 📌 Features

* User Signup and Login system
* Product browsing interface
* Add to cart functionality
* Responsive and clean user interface
* Interactive pages using JavaScript

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity and functionality

---

## 📄 Pages Overview

### Home Page

Displays products and allows users to browse items.

### Login Page

Allows registered users to log into their account.

### Sign Up Page

Allows new users to create an account.

### Cart Page

Displays products added to the shopping cart.

---

## 🚀 How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Run **homePage.html** in any web browser
